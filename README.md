# Marketing Website

A high-end, modern portfolio/marketing website featuring advanced animations and a sleek design.

## 🚀 Features
- **Modern UI**: Clean, Apple-inspired aesthetics with a focus on typography and whitespace.
- **Dynamic Animations**: Powered by GSAP (GreenSock Animation Platform) for smooth transitions and interactive elements.
- **Responsive Design**: Fully optimized for various screen sizes.
- **Glassmorphism**: Elegant use of blur effects and gradients.

## 🛠️ Built With
- **HTML5**: Semantic structure.
- **CSS3**: Custom styling and layout.
- **JavaScript**: Core logic and interactivity.
- **GSAP**: Industry-standard animation library.

## 📂 Project Structure
- `index.html`: Main entry point.
- `CSS/`: Stylesheets.
- `JS/`: Application logic.
- `Images/`: Optimized assets.

## 🔧 Local Development
Simply open `index.html` in any modern web browser or use a local live server.
